# 🌸 Task-2: Iris AI Classifier

A Machine Learning classification project built using the famous Iris Flower Dataset.

The project uses the K-Nearest Neighbors (KNN) algorithm to classify flowers into:

- Setosa
- Versicolor
- Virginica

### Features

- Dataset Exploration
- Data Visualization
- Feature Scaling
- Train-Test Split
- KNN Classification
- Hyperparameter Tuning
- Confusion Matrix
- Classification Report
- Interactive Prediction System

### Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn
